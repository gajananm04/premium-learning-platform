# 🚀 Quick Start Guide

## Installation

```bash
cd premium-learning-platform
npm install
npm run dev
```

Open: **http://localhost:5173**

---

## 🔐 Login

### Admin Login
1. Select "Admin" role
2. Click "Sign In"
3. Access full dashboard

### Student Login
1. Select "Student" role
2. Click "Sign In"
3. Access personal dashboard
